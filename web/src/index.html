The Little Computer 3 (LC-3) is a 16-bit computer architecture, devised for educational purposes by Yale N Patt.

kladdvara is a mostly-accurate emulator for an LC-3 computer equipped with a teletype terminal; it should run most text-based programs just fine. It was written for both in Rust for both native and web, compiled to WebAssembly via wasm-pack, and scaffolded for this web app with TypeScript via Webpack.
